### Updating the component

To get the latest version of the component, you can login 
into the client's area at https://webix.com/clients/

There you can download the latest package, or register as a Webix developer
and get npm access to the latest and all further versions of the package.  

To get npm access to Webix UI library, run the following command:

npm login --registry=http://npm.webix.com --scope=@xbs

When asked for credentials, use the data below:

User: jl1112
Password: 909a7eeb

After that, you can use the common npm commands to install Webix POR and Complex 
Widgets according to your license.

npm i @xbs/webix-pro
npm i @xbs/pivot
npm i @xbs/kanban
npm i @xbs/filemanager
npm i @xbs/scheduler
npm i @xbs/spreadsheet
npm i @xbs/querybuilder
